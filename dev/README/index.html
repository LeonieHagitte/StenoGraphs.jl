StenoGraphs.jl ― Write meta graphs quickly

Stenography: a quick way of writing using special signs or abbreviations

StenoGraphs.jl lets you quickly write meta graphs. As with shorthand, it is optimized for writing quickly (by humans) but is less quickly read (by computers).

To install StenoGraphs.jl:

import Pkg; Pkg.add(url="https://github.com/aaronpeikert/StenoGraphs.jl.git")

Your first @StenoGraph using StenoGraphs:

using StenoGraphs
@StenoGraph a → b
DirectedEdge{SimpleNode{Symbol}, SimpleNode{Symbol}}(SimpleNode{Symbol}(:a), SimpleNode{Symbol}(:b))

Multiple Nodes

Multiple nodes on one side lead to multiple edges:

@StenoGraph [a b] → c
2-element Vector{DirectedEdge{SimpleNode{Symbol}, SimpleNode{Symbol}}}:
 DirectedEdge{SimpleNode{Symbol}, SimpleNode{Symbol}}(SimpleNode{Symbol}(:a), SimpleNode{Symbol}(:c))
 DirectedEdge{SimpleNode{Symbol}, SimpleNode{Symbol}}(SimpleNode{Symbol}(:b), SimpleNode{Symbol}(:c))

Multiple nodes on both sides lead to the cross product of edges:

@StenoGraph [a b] → [c d]
4-element Vector{DirectedEdge{SimpleNode{Symbol}, SimpleNode{Symbol}}}:
 DirectedEdge{SimpleNode{Symbol}, SimpleNode{Symbol}}(SimpleNode{Symbol}(:a), SimpleNode{Symbol}(:c))
 DirectedEdge{SimpleNode{Symbol}, SimpleNode{Symbol}}(SimpleNode{Symbol}(:a), SimpleNode{Symbol}(:d))
 DirectedEdge{SimpleNode{Symbol}, SimpleNode{Symbol}}(SimpleNode{Symbol}(:b), SimpleNode{Symbol}(:c))
 DirectedEdge{SimpleNode{Symbol}, SimpleNode{Symbol}}(SimpleNode{Symbol}(:b), SimpleNode{Symbol}(:d))

Unless you specifically broadcast:

@StenoGraph [a, b] .→ [c, d]
2-element Vector{DirectedEdge{SimpleNode{Symbol}, SimpleNode{Symbol}}}:
 DirectedEdge{SimpleNode{Symbol}, SimpleNode{Symbol}}(SimpleNode{Symbol}(:a), SimpleNode{Symbol}(:c))
 DirectedEdge{SimpleNode{Symbol}, SimpleNode{Symbol}}(SimpleNode{Symbol}(:b), SimpleNode{Symbol}(:d))

Modification

Modification is done by overloading '*' for types of Modifier.

Let's define a Modifier:

struct Weight <: Modifier
    w::Number
end

A modifier can be directly applied to edges:

@StenoGraph (a → b) * Weight(1)
ModifiedEdge{DirectedEdge{SimpleNode{Symbol}, SimpleNode{Symbol}}, Vector{Main.Weight}}(DirectedEdge{SimpleNode{Symbol}, SimpleNode{Symbol}}(SimpleNode{Symbol}(:a), SimpleNode{Symbol}(:b)), Main.Weight[Main.Weight(1)])

Multiplying a Node with a Modifier leads to a ModifyingNode.

@StenoGraph b * Weight(1)
ModifyingNode{SimpleNode{Symbol}, Vector{Main.Weight}}(SimpleNode{Symbol}(:b), Main.Weight[Main.Weight(1)])

A ModifyingNode will modify its edges:

@StenoGraph a → b * Weight(1)
ModifiedEdge{DirectedEdge{SimpleNode{Symbol}, SimpleNode{Symbol}}, Vector{Main.Weight}}(DirectedEdge{SimpleNode{Symbol}, SimpleNode{Symbol}}(SimpleNode{Symbol}(:a), SimpleNode{Symbol}(:b)), Main.Weight[Main.Weight(1)])

To modify Nodes directly to create a ModifyingNode (instead of ModifyingNode), the following syntax is planned, but not implemented:

a^modifier()

Related Software

The R programming language has formulas of the form a ~ b to specify regressions. This inspired Yves Rosseel to create a very concise, yet expressive syntax for Structural Equation Models for lavaan. Stenographs.jl tries to maintain the best features of this syntax while creating Julia Objects that represent a graph (i.e., similar to MetaGraphs).
